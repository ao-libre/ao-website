Argentum Online Libre - Re-Launch

AO Libre is a collaborative project using open-source as a banner. Everyone is invited to participate by making modifications at the code level, graphics, sounds, adding items, spells, mapping or being part of the organization of the game within the server. All our resources and code are released under the AGPL license and can be found on Github. The project has more than 20 years of development and is still growing. It was always a dream to create a game that was free, open-source and free for everyone to play. Now we try to fund the project to work with professionals and make more features and alternative games of the metaverse possible.

This is an opportunity to re-launch and invite many people as possible to be part of the ecosystem of AO Libre project. We have plans to create a DAO for the project. Please join us!

Liquidity Generator Token type

Every transaction incurs a fee of 7% to the benefit of all holders, of which: 2% are distributed among all diamond hands holders. 3% are locked away in the liquidity pool, to create a steadily rising price floor. 2% is distributed to marketing.
